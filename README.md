# RSA Encryption Program

A C++ implementation of the RSA cryptography algorithm built using object-oriented programming principles. This project demonstrates secure message encryption and decryption using public and private key generation.

## Features

- RSA key generation
- Message encryption
- Message decryption
- Modular arithmetic implementation
- Object-oriented design in C++

## Technologies Used

- C++
- Object-Oriented Programming
- Number Theory
- Cryptography Concepts

## How It Works

The program:

1. Generates public and private RSA keys
2. Encrypts plaintext messages using the public key
3. Decrypts ciphertext using the private key

This project was built to strengthen understanding of:
- Encryption algorithms
- Prime numbers and modular arithmetic
- Secure data handling
- OOP design patterns

## Example

```bash
Enter message: hello
Encrypted message: [ciphertext]
Decrypted message: hello
```

## Installation

Clone the repository:

```bash
git clone https://github.com/basetsowab/Rsa_project_Baset_Sowab.git
```

Compile the program:

```bash
g++ main.cpp -o rsa
```

Run the executable:

```bash
./rsa
```
